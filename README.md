🛒 AI-Based Customer Segmentation
📌 Project Overview

This project focuses on Customer Segmentation using Unsupervised Machine Learning techniques on the Wholesale Customers Dataset.

The objective is to group customers based on their spending behavior and analyze patterns using clustering algorithms.

This was developed as a group project as part of our AI training program.

🧠 Problem Statement

Wholesale distributors serve different types of customers.
Understanding customer purchasing patterns helps businesses:

Identify high-value customers

Optimize marketing strategies

Improve inventory planning

Personalize business decisions

We used clustering techniques to segment customers based on their annual spending in different product categories.

📊 Dataset Information
Dataset: Wholesale Customers Dataset
Source: Kaggle
Features:
Fresh
Milk
Grocery
Frozen
Detergents_Paper
Delicassen

Each row represents a customer’s annual spending in different categories.

⚙️ Machine Learning Techniques Used

1️⃣ K-Means Clustering

Used Elbow Method to determine optimal clusters
Grouped customers based on spending similarity
Efficient and scalable clustering approach

2️⃣ Hierarchical Clustering

Used Agglomerative approach
Visualized clusters using Dendrogram
Helped understand hierarchical relationships between customers

✔ Data Preprocessing
